# BI Application Release

## Descripción del Proyecto
Este proyecto tiene como objetivo construir una aplicación de Business Intelligence (BI) enfocada en la simulación y validación de inventarios, análisis de riesgos y comunicación estratégica de insights para validar la introducción de un nuevo producto al mercado. El repositorio integra procesos de modelado de datos, visualización y documentación para soportar decisiones empresariales con información confiable y clara.

## Objetivos de BI
- Diseñar modelos de datos realistas para inventarios y ventas.
- Implementar KPIs y KRIs basados en frameworks reconocidos.
- Construir dashboards dinámicos en Power BI que permitan comparar escenarios simulados vs. reales.
- Validar resultados con datos externos de mercado para asegurar benchmarking confiable.
- Comunicar hallazgos de manera ejecutiva con enfoque en impacto estratégico.

## Herramientas Utilizadas
- **Git & GitHub**: Control de versiones y colaboración.
- **Power BI**: Visualización de KPIs, dashboards interactivos.
- **dbt (Data Build Tool)**: Transformaciones SQL y modelado modular.
- **PostgreSQL**: Base de datos para almacenamiento y consultas.
- **BigQuery (opcional)**: Integración en la nube para escalabilidad y benchmarking.
- **Python**: Desarrollo de modelos analíticos avanzados:
  - Forecasting de ventas y demanda.
  - Pipelines de decisiones para escenarios estratégicos.
  - Análisis y simulación financiera.
- **SQL**: Consultas y validación de datos.
- **Power Query**: Limpieza y transformación de datos.
